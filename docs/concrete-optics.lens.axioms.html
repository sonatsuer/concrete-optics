concrete-optics.lens.axioms

Lens axioms.

get-put-axiom

(get-put-axiom optic whole part & [comparison-function])

Checks the lens axiom which states put composed with view gives you the original part you put. If a comparison function is not provided then typed-eq is used.

put-put-axiom

(put-put-axiom optic whole part_1 part_2 & [comparison-function])

Checks the lens axiom which states if you put two values in a whole then the second one overrides the first one. If a comparison function is not provided then typed-eq is used.
